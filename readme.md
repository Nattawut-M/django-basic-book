# Book Store  
เว็บแอพพลิเคชันร้านหนังสือออนไลน์

## Stack

1. Django Framework 3.1.x
2. Bootstrap 4 (CDN)
3. Sqlite

## Installation

1. activate enviroment (mac os)  `source venv/bin/activate`
2. install package `pip3 install -r requirement.txt`
3. run django   `python3 manage.py runserver`
